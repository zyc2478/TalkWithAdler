# 微信小程序域名配置指南

## 问题分析
微信小程序在真机环境中需要在后台配置合法的域名，否则无法访问外部API。

## 解决方案

### 方法一：在微信公众平台配置域名（推荐）
1. **登录微信公众平台**：https://mp.weixin.qq.com/
2. **进入小程序管理**：选择你的小程序
3. **开发设置**：在左侧菜单中找到"开发设置"
4. **服务器域名**：
   - 在"request合法域名"中添加：`https://dashscope.aliyuncs.com`
   - 保存设置
5. **等待生效**：域名配置需要10-15分钟生效

### 方法二：使用云函数作为代理
1. **创建云函数**：在小程序中创建一个云函数
2. **实现代理逻辑**：云函数调用DashScope API，小程序调用云函数
3. **部署云函数**：将云函数部署到微信云开发环境

### 方法三：使用已配置的API服务
如果已经有其他合法域名的API服务，可以通过该服务转发请求到DashScope API。

## 验证步骤
1. 配置完成后，在微信开发者工具中重新编译项目
2. 在真机上测试对话功能
3. 检查是否能正常连接大模型API

## 注意事项
- 域名配置需要小程序的管理员权限
- 配置后需要等待一段时间才能生效
- 确保API Key的安全性，不要在前端代码中直接暴露
